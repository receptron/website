# dataSumTemplateAgent

## Description

Returns the sum of input values

## Schema

#### inputs

```json
{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "the array of numbers to calculate the sum of",
      "items": {
        "type": "integer"
      }
    }
  },
  "required": ["array"]
}
```

#### output

```json
{
  "type": "number"
}
```
