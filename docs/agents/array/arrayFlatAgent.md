# arrayFlatAgent

## Description

Array Flat Agent

## Schema

#### inputs

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "flat array"
    }
  },
  "required": [
    "array"
  ]
}

````

#### output

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "the remaining array"
    }
  }
}

````
